# Research

Research documents live on the [`docs/research`](https://github.com/ridermw/nanopilot/tree/docs/research/research) branch to keep `main` focused on the core runtime.

This is a **static content branch** — it is not forward-merged with main like `skill/*` branches. Research files have no code dependencies.

## Browse

View the full research catalog (INDEX.md and 16 topic files) by switching to the branch:

```bash
git fetch origin docs/research
git checkout origin/docs/research -- research/
```

Or browse online: https://github.com/ridermw/nanopilot/tree/docs/research/research
